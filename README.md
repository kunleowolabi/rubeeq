# Rubeeq

AI-powered extraction engine that converts any exam PDF into structured,
database-ready data.

> **Proprietary software.** Source is visible for review purposes only.
> See [LICENSE](./LICENSE) for terms.

---

## What it does

Feed it a questions paper and marking scheme — from any exam board, in any
format — and it returns a SQL schema, a JSON data bundle, and a ready-to-run
insert script. It handles native and scanned PDFs, identifies the exam type
automatically, and extracts questions, mark allocations, and marking schemes
with full fidelity.
